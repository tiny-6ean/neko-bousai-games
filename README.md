# ねこ防災ミニゲーム｜静灯工房

猫と暮らす人のための  
静かでシンプルな防災学習ミニゲーム集です。

避難時の行動や備えを  
落ち着いて確認できるように設計しています。

---

## 🎮 ミニゲーム一覧

### 🐾 猫を探すミニゲーム
災害時に猫が隠れやすい場所を学べるミニゲームです。  
性格・年齢・健康状態に応じて、隠れやすい場所が変化します。

### 🎒 避難袋チェックミニゲーム
猫のプロフィールに合わせて  
避難袋の中身を優先度ベースで確認できます。  
高・中・低の優先度と、各アイテムのプチアドバイス付き。

---

## 🌿 コンセプト

静灯工房の世界観である  
**淡色・静けさ・余白**を大切にした  
ミニマルな学習ツールです。

- 1ページで完結する軽量設計  
- スマホでも読みやすい UI  
- 実用性と優しさを両立した防災学習  
- 猫の性格・年齢・健康状態に合わせた判定
