# ねこ防災ミニゲーム｜静灯工房

猫と暮らす人のための  
静かでシンプルな防災学習ミニゲーム集です。

災害時の備え・行動・ケアを  
落ち着いて確認できるように設計しています。

---

## 🎮 ミニゲーム一覧

### 🐾 猫を探すミニゲーム
災害時に猫が隠れやすい場所を学べるミニゲームです。  
性格・年齢・健康状態に応じて、隠れやすい場所が変化します。

### 🎒 避難袋チェックミニゲーム
猫のプロフィールに合わせて  
避難袋の中身を優先度ベースで確認できます。  
高・中・低の優先度と、各アイテムのプチアドバイス付き。

### 💊 災害時ケアミニゲーム
災害時に必要となる猫のケアを  
状況に応じて静かに整理できるミニゲームです。  
体調・環境・ストレスなど複数の視点から確認できます。

---

## 🌿 コンセプト

静灯工房の世界観である  
**淡色・静けさ・余白**を大切にした  
ミニマルな学習ツールです。

- 1ページで完結する軽量設計  
- スマホでも読みやすい UI  
- 実用性と優しさを両立した防災学習  
- 猫の性格・年齢・健康状態に合わせた判定  
- 災害時の行動・備え・ケアを静かに整理できる構成  
- ミニゲームの追加に柔軟に対応できる設計
